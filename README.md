# projeto_final_mack_data_collection_e_storage
Repositório para armazenamento dos arquivos correspondentes à entrega do trabalho final da disciplina Data Collection e Storage do MBA de Engenharia de Dados.

#Arquivos
Neste repositório, estão disponibilizados dois arquivos: o arquivo "entrega_final_data_storage_collection.ipynb", contendo todos os códigos correspondentes à execução das funções de otimização, particionamento, compressão, escrita e leitura do dataset de teste, e o arquivo "Trabalho Final (TEMA B) - DATA COLLECTION E STORAGE", contendo o resumo executivo e as conclusões tiradas a partir da execução do código.

#Instruções para execução do código
É necessário estabelecer a conexão do Google Colab com o runtime padrão da ferramenta, e então executar as células na ordem em que estão escritas no arquivo.
